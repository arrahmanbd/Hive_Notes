# Hive_Notes
Note Takig App Using hive_db

## Installation
After cloning this repository, Run pub get commands, or add dependency:
```bash
  hive: ^2.2.2
  hive_flutter: ^1.1.0
  
```
```bash
  path_provider: ^2.0.11
```
